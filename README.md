# Ottoman Tasbih Mobile Store

Mid-Term project mobile application submission for Asst. Prof. Dr. Deniz DAHMAN.

This is a cross-platform mobile e-commerce application built using the Flutter framework and Dart. It features stateful navigation, a dynamic shopping cart UI, and an interactive account creation screen.

## How to Run This Project

To run this app on your local machine, please follow these steps:

1. Prerequisites: Ensure you have the [Flutter SDK](https://docs.flutter.dev/get-started/install) installed on your computer.
2. Download: Clone or download this repository to your local machine.
3. Open: Open the `ottoman_tasbih` folder in Android Studio or Visual Studio Code.
4. Install Packages: Open the terminal inside the editor and run the following command to install the necessary dependencies:
   ```bash
   flutter pub get
